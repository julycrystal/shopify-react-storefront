# Shopify Storefront With Custom Styles

## Stacks used

### For Shopify integration

- Remix
- Hydrogen
- Oxygen
- Shopify CLI

### For Developer Experience

- Typescript
- ESLint
- Prettier
- GraphQL generator
- GraphQL Codegen
- Concurrently for easy startup

### For styling

- Tailwindcss
- React Chartjs 2

## Getting started

Requirements:

- Node.js version 16.14.0 or higher

### Building for production

```
npm run build
```

### Local development

```
npm run dev
```
